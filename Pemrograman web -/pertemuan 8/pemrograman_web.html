HTML



HHTML adalah tag yang digunakan sebagai penanda atau kerangka dalam web contohnya seperti text, audio , atau pun video

CSS



CSS adalah script-script yang digunakan untuk memperindah tampilan -Website contohnya dan pengaturan warna , text , dan layout sebuah web

Javascript


JavaScript adalah bahasa pemrgraman yang dapat membuat website menjadi interaktif, contohnya pengaturan input output serta animasi
